## About

**ScooterNav (Scooter Pilot)** is a Progressive Web App (PWA) purpose-built for electric scooter riders. Unlike generic GPS apps, ScooterNav is optimized for micro-mobility — routing riders through protected bike lanes, alerting them to hazards, locating nearby EV charging stations, and providing turn-by-turn voice navigation in 9 languages. It works offline, installs on any device like a native app, and is completely free to use.

Developed by **M&S.co**, founded by Shane Matthew Russell and Manuel Hernandez.

## Features

| Feature | Description |
|---|---|
| **Bike Lane Routing** | Automatically detects and routes through protected, shared, or standard lanes |
| **EV Charging Station Finder** | Locates nearby charging stations with proximity notifications |
| **Offline Maps** | Caches map data locally for connectivity-independent navigation |
| **Voice Guidance** | Turn-by-turn audio directions in 9 languages |
| **Multi-Destination Planning** | Plan routes with multiple stops |
| **Real-Time Safety Alerts** | Live weather and road hazard monitoring |
| **Dark / Light Mode** | User-selectable interface theme |
| **PWA Installation** | Install directly from the browser on any device |
| **Intelligent Routing** | Geodesic calculations for optimal scooter-friendly paths |

## Installation

### Use it instantly (no install required)

Visit **[https://scooter-pilot--shanemrussell88.replit.app](https://scooter-pilot--shanemrussell88.replit.app)** in any modern browser.

### Install as a PWA

**On mobile (iOS / Android):**
1. Open the link above in your browser
2. Tap **Share** → **Add to Home Screen**

**On desktop (Chrome / Edge):**
1. Open the link above
2. Click the install icon (⊕) in the address bar
3. Click **Install**

### Run locally (from source)

```bash
# Clone the repository
git clone https://github.com/shanemrussell888-coder/Scooter-Pilot.git

cd Scooter-Pilot

# Serve with any static server:
npx serve .
# or
python -m http.server 8080
